## ğŸ› JIKA ADA MASALAH

### Error: "Firebase config not found"
**Solusi**: Game masih bisa dimainkan offline, tapi skor tidak tersimpan. Isi `firebase-config.js` dengan credentials Firebase Anda.

### Error: "Dashboard kosong"
**Solusi**: 
1. Mainkan game sampai selesai
2. Tunggu 1-2 menit untuk Firebase sync
3. Refresh dashboard.html
4. Buka browser console (F12) untuk melihat error

### Musik tidak bunyi
**Solusi**: Klik tombol di game terlebih dahulu (browser perlu user interaction untuk audio)

### Skor tidak tersimpan
**Solusi**:
1. Cek `firebase-config.js` sudah diisi dengan credentials
2. Cek Firestore "test mode" sudah enabled
3. Buka browser console (F12) â†’ lihat error
4. Cek network tab â†’ lihat request ke Firebase
