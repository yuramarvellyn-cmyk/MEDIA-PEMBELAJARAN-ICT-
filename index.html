‚ùì Pertanyaan Umum

Apakah game ini gratis?

Ya! Physics Adventure 100% gratis untuk dimainkan. Tidak ada biaya tersembunyi atau in-app purchase.

Bisakah saya bermain offline?

Ya! Game bisa dimainkan offline. Tapi skor tidak akan tersimpan dan dashboard tidak akan update. Untuk fitur online, setup Firebase terlebih dahulu.

Berapa lama main satu game?

Rata-rata 15-25 menit tergantung kecepatan menjawab kuis. Anda bisa ulangi kapan saja untuk mencari skor lebih tinggi.

Apakah cocok untuk pelajar?

Ya! Game ini dirancang khusus untuk pelajar SMA yang sedang belajar Momentum & Impuls. Setiap level menjelaskan konsep fisika dengan visual menarik.

Bagaimana jika jawaban saya salah?

Tidak apa-apa! Anda akan kehilangan 5 poin dan bisa mencoba lagi. Setiap kuis disertai penjelasan untuk membantu belajar.
